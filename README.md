# WELCOME

This is my personal site. I've tried several different options and settled on using GitHUb for the ease of updating compared to something like netlify. 

The architecture is a bit unusual, but it suits me. Since I'm used to writing in Markdown, specifically in obsidian, I styled the site around that. 

Each .html is essentially a shell with some scripts that points to a .md file of the same name. This .md file contains all of the actual content, including links. A script is ran that translates the .md into .html. 

This was vibecoded with claude for the most part, but my creativity still plays a large part. I tried to set it up with expansion in mind.
